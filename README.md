# SAC AR-CV VEN (Demo estable para GitHub Pages)

## Publicar en GitHub Pages (fácil)
1) Sube este proyecto completo a un repo (rama `main`)
2) En GitHub: Settings → Pages → Source = **GitHub Actions**
3) Ve a la pestaña **Actions** y espera que el workflow termine en ✅ (verde)
4) Abre tu URL de Pages

## Notas
- El chatbot está en **modo demo**, no usa API keys y **no rompe** la web.
- Vite está configurado con `base: "./"` para que funcione en sub-rutas (GitHub Pages).
